Dycapo Server Test

Welcome to the official Dycapo Server Test site.

In this site you can test some Dycapo Protocol calls, to be made against

http://test.dycapo.org/RPC2

Some XML-RPC implementations require that you omit the /RPC2 part.
Please see the README file to know which username and password to use. As this is the test server used for development, we can neither ensure a decent uptime, nor great performances, nor database data stability, nor compatibility to the stable APIs and Protocol!

Due to restrictations of our hosting service, this server does not run in HTTPS. Therefore be aware to what you send through the network!

Dycapo Server implementation is heavily documented, therefore download and run your own copy of it instead of bombing this tiny server. Please use the links on top of this page to learn more about the project and the Protocol.
